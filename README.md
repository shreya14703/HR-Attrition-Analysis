# 📊 HR Attrition Analysis Dashboard – Excel Project

## 📌 Project Overview
This project presents an interactive **HR Attrition Analysis Dashboard** built in **Microsoft Excel** to analyze employee turnover patterns and identify key factors influencing attrition.

The dashboard converts raw HR data into actionable insights using **Pivot Tables, Pivot Charts, Slicers, and Data Visualization techniques**, enabling better workforce decision-making.

---

## 🎯 Objectives
- Analyze employee attrition across departments, job roles, and demographics  
- Identify factors affecting attrition such as gender, education, income, and age  
- Visualize employee trends using interactive charts  
- Enable dynamic filtering using **Slicers**  
- Provide insights to improve employee retention strategies  

---

## 📁 Dataset Description
The dataset includes employee-level HR information:

| Column Name        | Description                                      |
|-------------------|--------------------------------------------------|
| Employee ID       | Unique identifier for each employee              |
| Department        | Department of the employee                       |
| Job Role          | Role/designation of the employee                 |
| Gender            | Gender of the employee                           |
| Age               | Age of the employee                              |
| Education         | Education level                                  |
| Monthly Income    | Salary of the employee                           |
| Attrition         | Employee status (Yes = Left, No = Stayed)        |

---

## 📊 Dashboard Features

### 1. Attrition Analysis
- **Attrition by Department** – Highlights departments with high employee turnover  
- **Attrition by Job Role** – Shows which roles have the highest attrition  
- **Attrition by Age** – Analyzes attrition distribution across age groups  

---

### 2. Demographic Analysis
- **Attrition by Gender** – Compares attrition between male and female employees  
- **Attrition by Education** – Shows impact of education level on attrition  

---

### 3. Salary Analysis
- **Attrition by Monthly Income** – Identifies relationship between salary and employee exits  

---

### 4. Advanced Filtering (Slicers)
Interactive slicers allow filtering by:
- Department  
- Gender  
- Job Role  
- Education  
- Attrition Status  

This enables detailed and customized analysis.

---

## 🔍 Key Insights
- Research-related departments show higher attrition compared to others  
- Female employees have slightly higher attrition compared to male employees  
- Attrition varies significantly across different job roles  
- Employees with lower income are more likely to leave  
- Most employees remain in the organization, indicating stable retention  

---

## 💡 Business Recommendations
- Focus retention strategies on high-attrition departments  
- Review salary structures for lower-income employees  
- Improve job satisfaction and growth opportunities in critical roles  
- Promote diversity and employee engagement programs  
- Provide targeted training and career development plans

## 🚀 How to Use
1. Open the Excel file in **Microsoft Excel (2016 or later)**  
2. Go to the **Dashboard sheet**  
3. Use **Slicers** to filter by Department, Gender, Job Role, and Education  
4. Observe dynamic updates in charts  
5. Analyze trends and insights for decision-making
